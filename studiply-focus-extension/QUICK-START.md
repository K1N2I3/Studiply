# 🚀 快速开始 - Studiply Focus Mode Extension

## ✅ 问题已解决！

图标问题已经解决，扩展程序现在可以正常安装和使用。

## 📦 安装步骤

### 1. 安装扩展程序
1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"（右上角开关）
4. 点击"加载已解压的扩展程序"
5. 选择 `studiply-focus-extension` 文件夹
6. ✅ 扩展程序安装成功！

### 2. 使用Focus Mode
1. 访问 [Studiply.it](https://studiply.it)
2. 进入Focus Mode页面
3. 看到扩展程序状态指示器：
   - 🟢 绿色 = 扩展程序已激活
   - 🔵 蓝色 = 扩展程序已安装，等待激活
   - 🟠 橙色 = 扩展程序未安装
4. 点击"Start"开始专注会话
5. 扩展程序自动激活，开始阻止分散注意力的网站

## 🔧 故障排除

### 扩展程序没有工作？
- 检查扩展程序是否已启用
- 刷新Studiply网站
- 重新启动浏览器

### 网站没有被阻止？
- 确认扩展程序状态显示为"Extension Active"
- 检查被阻止的网站列表
- 尝试手动刷新被阻止的网站

### 需要图标？
- 参考 `ICON-SETUP.md` 文件
- 或直接使用，扩展程序无需图标也能正常工作
