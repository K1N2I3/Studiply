# 🎯 Studiply Focus Mode Extension - 安装指南

## 快速安装步骤

### 1. 下载扩展程序
- 下载整个 `studiply-focus-extension` 文件夹到你的电脑

### 1.5. 解决图标问题（如果遇到错误）
如果安装时遇到 "Could not load icon" 错误：
- 扩展程序已经配置为无需图标即可工作
- 如果需要图标，请参考 `ICON-SETUP.md` 文件
- 或者直接跳过此步骤，扩展程序仍可正常使用

### 2. 安装到Chrome/Edge
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 在右上角开启"开发者模式"（Developer mode）
4. 点击"加载已解压的扩展程序"（Load unpacked）
5. 选择你下载的 `studiply-focus-extension` 文件夹
6. 扩展程序会出现在扩展列表中

### 3. 安装到Firefox
1. 打开Firefox浏览器
2. 在地址栏输入：`about:debugging`
3. 点击"此Firefox"（This Firefox）
4. 点击"临时载入附加组件"（Load Temporary Add-on）
5. 选择 `studiply-focus-extension` 文件夹中的 `manifest.json` 文件

## 使用方法

1. **安装完成后**，访问 [Studiply.it](https://studiply.it)
2. **进入Focus Mode页面**
3. **点击"Start"按钮**开始专注会话
4. **扩展程序会自动激活**，阻止分散注意力的网站
5. **尝试访问被阻止的网站**（如Facebook、Instagram等），会被重定向到提醒页面

## 功能特点

✅ **自动网站阻止** - 阻止Facebook、Instagram、Twitter等分散注意力的网站  
✅ **友好提醒页面** - 显示专注进度和统计信息  
✅ **实时状态同步** - 与Studiply网站实时同步状态  
✅ **会话进度显示** - 显示剩余时间和完成进度  
✅ **统计信息** - 显示被阻止的网站数量和专注时间  

## 被阻止的网站（默认）

- Facebook.com
- Instagram.com  
- Twitter.com
- TikTok.com
- YouTube.com

- Snapchat.com

## 扩展程序状态指示器

在Focus Mode页面中，你会看到扩展程序状态指示器：

- 🟢 **绿色** - 扩展程序已激活，正在阻止网站
- 🔵 **蓝色** - 扩展程序已安装，等待激活
- 🟠 **橙色** - 扩展程序未安装，建议安装以获得完整功能

## 故障排除

### 扩展程序没有工作？
1. 检查扩展程序是否已启用
2. 刷新Studiply网站
3. 重新启动浏览器
4. 检查浏览器控制台是否有错误信息

### 网站没有被阻止？
1. 确认扩展程序状态显示为"Extension Active"
2. 检查被阻止的网站列表
3. 尝试手动刷新被阻止的网站

### 扩展程序无法与网站通信？
1. 确认你在Studiply.it网站上
2. 检查浏览器是否允许扩展程序访问网站
3. 重新加载扩展程序

## 隐私说明

- 扩展程序只在专注会话期间激活
- 不会收集或存储个人数据
- 所有数据都存储在本地浏览器中
- 只与Studiply网站通信

## 支持

如果遇到问题，请：
1. 检查浏览器控制台错误信息
2. 确认扩展程序权限设置
3. 尝试重新安装扩展程序
